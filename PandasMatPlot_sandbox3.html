###x = pd.to_datetime(df['Time'])
x = df['Time']
y1 = df['%user']
y2 = df['%system']
y3 = df['%iowait']
y4 = df['%idle']

# plot
plt.plot(x,y1)
plt.plot(x,y2)
plt.plot(x,y3)
plt.plot(x,y4)
# beautify the x-labels
plt.gcf().autofmt_xdate()
plt.ylabel('Percent')
plt.xlabel('Time')
plt.legend(loc=0,prop={'size': 8})
plt.xticks(rotation='vertical', fontsize = 6)
###plt.rcParams['figure.figsize'] = [15, 15]
plt.rcParams['figure.dpi'] = 300
plt.show()

Not that a scatter chart makes sense with this data, I just wanted to see how it looks and how easy it would be to change graph types.
